# 雏雁计划 · 智能助教系统插件

> 基于 LLM 的跨学科启发式智能助教插件，面向课后习题答疑与学情沉淀。  
> 适配 AstrBot，支持图文混合输入、过程化引导回复与本地学习数据持久化。

---

## 项目简介

本插件用于构建“雏雁计划”课后习题智能助教场景：  
学生可通过文字或图片提问，助教模型对题目进行分析与引导，强调思路启发而非直接给出完整答案，并将关键交互数据落盘，便于后续学情分析与服务迭代。

---

## 核心特性 Features

### 1) 图文多模态题目解析
- 同时解析学生发送的文字与图片链接
- 支持仅图片提问、仅文字提问、图文混合提问
- 自动将输入统一送入 LLM 进行题意理解与答疑

### 2) 启发式引导（拒绝直接给答案）
- 助教人格强调“讲思路、给提示、做纠偏”
- 优先引导学生自我推理，避免一步到位给出最终答案
- 更适配课后练习与课堂巩固的教学目标

### 3) 内置 SQLite 学情数据持久化
- 内置异步数据库初始化与写入流程
- 自动创建 `tutor_records` 表并保存问答记录
- 当前采用 SQLite，本地调试便捷，后续可平滑迁移 PostgreSQL

## 安装指南 Installation

### 1) 放置插件目录
将插件目录放入 AstrBot 插件路径：  
`data/plugins`

建议目录结构如下（示意）：

```text
astrbot_plugin_smart_tutor/
├─ main.py
├─ requirements.txt
└─ README.md
```

### 2) 安装依赖（关键）
在 AstrBot 运行环境中执行以下任一方式：

方式 A：

```bash
pip install aiosqlite
```

方式 B：

```bash
pip install -r requirements.txt
```

## 使用说明 Usage

### 触发命令
- `/助教`
- `/tutor`

### 基本用法
- 文字提问：在命令后直接输入题目
- 图片提问：发送命令并附上题目截图
- 图文提问：命令 + 文字描述 + 图片同时发送

### 示例（带图片提问）
学生发送：

```text
/助教 这是我今天的函数题，麻烦帮我看看思路
（同时附上一张题目截图）
```

插件返回（示例风格）：
- 先提炼题目考点
- 再指出常见误区
- 最后给出下一步提示，而不是直接公布完整答案

---

## ⚠️ 注意事项 Important Notes

> **必须在 AstrBot 面板中配置支持视觉（Vision）的大模型**，例如 `gpt-4o`、`qwen-vl-plus`。  
> **并且必须在模型设置中关闭 Function Calling / 工具调用功能，否则将无法识别图片。**

如果未满足以上条件，插件可能出现以下现象：
- 图片输入无法被理解
- 回答仅基于文本，忽略题图内容
- 多模态问答效果显著下降

## 数据表说明（当前版本）

表名：`tutor_records`

字段：
- `id`：自增主键
- `user_id`：学生标识（优先 QQ 号，其次群标识）
- `question_content`：学生问题内容（文本与图片链接序列化）
- `bot_reply`：助教回复内容
- `timestamp`：记录时间（默认当前时间）
